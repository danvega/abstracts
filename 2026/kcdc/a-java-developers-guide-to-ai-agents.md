# A Java Developers Guide to AI Agents

**Format:** Session

## Abstract

AI isn't just about answering questions anymore. It's about getting things done. The developers who understand how to build agents and agentic workflows won't just be writing better software, they'll be building systems that work for them. If you're a Java developer who wants to stay ahead of where the industry is heading, this is the session for you.

You already know how to integrate AI into your Java applications. Now it's time to go further. In this talk we cover MCP, the protocol that has become the standard for connecting AI to the real world. We explore how skills give your agents real capabilities, how agentic workflows enable your application to reason and act across multiple steps, and how to bring it all together into autonomous agents that get things done on your behalf. Everything we build is in Java using Spring AI.

Java has always been the language of enterprise systems. Now it's the language of intelligent ones too.
