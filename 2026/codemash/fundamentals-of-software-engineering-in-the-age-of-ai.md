# Fundamentals of Software Engineering in the Age of AI

**Track:** Programming Principles

**Format:** Session (60 min)

## Abstract

Agentic IDEs and AI chat interfaces are transforming development workflows, leading some to proclaim the end of software engineering. Is it time for you to explore other careers? Not so fast, the rumors of our demise are greatly exaggerated. While these tools can generate boilerplate and enable "vibe coding," working code and engineered systems are worlds apart.

In this session, you'll discover why the fundamentals of software engineering are vital, not replaceable, in the age of AI. We'll explore the journey from programmer to engineer, covering core skills like reading and writing code effectively, technical practices including software modeling and working with existing codebases, and essential application development concepts from UI design to software architecture.

You'll learn how mastering soft skills to career management become even more critical when the task of writing boilerplate code is common amongst your peers. Based on our book "Fundamentals of Software Engineering," this talk will equip you with a roadmap for thriving in an AI-augmented development landscape and understanding what it truly means to evolve from writing code to being a software engineer.
