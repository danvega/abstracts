# What's New in Spring

**Track:** Java

**Format:** Conference (50 min)

## Elevator Pitch

Spring Boot 4 and Spring Framework 7 are the first major versions in years. This session covers what's new and what matters: null safety, built-in resilience, API versioning, improved testing, and more. You'll see practical examples and leave ready to upgrade.

## Abstract

Spring Boot releases every six months. Most updates are incremental. This one is different.

Spring Boot 4 and Spring Framework 7 are the first major versions in years. Null safety with JSpecify. Built-in resilience features. API versioning. Improved testing support. These features change how you write and maintain Spring applications.

In this session, we'll walk through the features that matter most. You'll see practical examples of each one so you understand not just what's new, but how to use it. By the end, you'll be ready to upgrade your applications and put these features to work.

## Key Takeaways

1. Understand the major new features in Spring Boot 4 and Spring Framework 7
2. See practical examples of null safety, resilience, API versioning, and improved testing
3. Know how to upgrade your existing applications to take advantage of these features
