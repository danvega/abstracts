# Getting Started with Spring AI

**Track:** Data

**Format:** Mini Lab (90 min)

## Elevator Pitch

Calling an LLM is easy. Building real AI features is harder. In this hands-on lab, you'll use Spring AI to connect to models, parse structured output, manage prompts, and wire up function calling. You'll leave with working code and the skills to integrate AI into your own Java applications.

## Abstract

You can call an LLM with a REST endpoint. But then what? You need to parse responses into objects. You need to swap models without rewriting code. You need the AI to call your Java methods. These are the problems Spring AI solves.

In this hands-on lab, you'll build AI features into a Java application using Spring AI. We'll start by connecting to a model and making your first call. Then we'll add structured output parsing to get real Java objects back. You'll learn how prompt templates keep your code clean. And you'll wire up function calling so the LLM can interact with your application.

By the end, you'll have working code and a clear path for integrating AI into your own Java applications.

## Key Takeaways

1. Understand why a framework like Spring AI matters for real AI integrations
2. Build AI features using structured output, prompt templates, and function calling
3. Leave with working code you can extend in your own applications
