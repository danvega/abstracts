# Claude Code: Start Simple, Scale Smart

**Track:** Development Practices

**Format:** Tools-in-Action (30 min)

## Elevator Pitch

Claude Code has a lot of features and it's easy to feel overwhelmed. This session shows you how to start simple and scale up. You'll learn the basics that matter, then see how custom slash commands, skills, and agents can make Claude Code truly yours.

## Abstract

Claude Code is powerful. Maybe too powerful. With so many features, it's hard to know where to start.

In this session, I'll show you how I actually use Claude Code day to day. We'll start with the basics that deliver immediate value. Then we'll build up to the features that make Claude Code truly yours: custom slash commands, skills, and agents that fit your workflow.

You don't need to learn everything at once. You need to know what matters first and what to add as you grow. By the end, you'll have a clear path from beginner to power user.

## Key Takeaways

1. Learn the Claude Code basics that deliver immediate value
2. Create custom slash commands, skills, and agents that fit your workflow
3. Build a clear path from beginner to power user at your own pace
